# Elroids Enterprise Pty Ltd (Landing Page)


## 💡 Technologies Used

  - ReactJS
  - Tailwindcss


## ⚙️ Run Locally

Clone the project

```bash
  git clone https://github.com/ethandiedericks/elroids-enterprise.git
```

Go to the project directory

```bash
  cd elroids-enterprise
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
